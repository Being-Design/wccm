== Description ==

Adds structured Google API location data to BuddyPress Profile fields and Groups, with address suggestion and autocomplete.

== Installation ==

1. Make sure BuddyPress is activated.
2. Visit 'Plugins > Add New'
3. Click 'Upload Plugin'
4. Upload the file 'buddypress-location-autocomplete.zip'
5. Activate 'Location Autocomplete for BuddyPress' from your Plugins page.

= Configuration =

1. Enable 'Extended Profiles' at 'Settings > BuddyPress > Components'
2. Create location Text Area profile fields at 'Users > Profile Fields'
2. Visit 'BuddyBoss > Location Fields' to match profile fields to their location sources.
